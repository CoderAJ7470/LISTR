LISTR - An idea I have for an app that allows you make lists. Essentially a to-do app with many more features.

**List** → List all items you want in different, itemized to-do lists
**Index** → tab reordering, pinned lists, quick-jump search
**Schedule** → weekly indicators, recurrence rules
**Track** → progress meters of some kind (circles/bars/squares - tbd), streaks, completion stats
**Remind** → soft nudges, visual alerts (with/without notifications - to be determined)

This side project is really just for me to get better at creating fullstack apps. As such, it will have a backend of some sort, user accounts and data persistence, to store/remove lists.
